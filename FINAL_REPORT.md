### ✨ Реализованный функционал

#### Основные функции:
- ✅ **Кирка Джека** - золотая зачарованная кирка с одноразовым использованием
- ✅ **Ломание спавнеров** - только разрешенных типов мобов
- ✅ **20% шанс дропа** - кастомный спавнер с сохраненным типом моба
- ✅ **Автоматическая активация** - установленный спавнер сразу работает
- ✅ **Команда /givejack** - выдача кирки (права: jackspawner.give)
- ✅ **Русская локализация** - все названия на русском языке

#### Поддерживаемые мобы:
- Зомби (ZOMBIE) → "Спавнер зомби"
- Скелет (SKELETON) → "Спавнер скелетов"
- Паук (SPIDER) → "Спавнер пауков"
- Пещерный паук (CAVE_SPIDER) → "Спавнер пещерных пауков"
- Ведьма (WITCH) → "Спавнер ведьм"
- Страж (GUARDIAN) → "Спавнер стражей"
- Ифрит (BLAZE) → "Спавнер ифритов"
- Чешуйница (SILVERFISH) → "Спавнер чешуйниц"

### 🎮 Установка и использование

1. Скачайте `JackSpawner-1.0.0.jar`
2. Поместите в папку `plugins/` сервера Paper 1.21.8
3. Запустите сервер
4. Используйте `/givejack` для получения кирки

### 💻 Сборка из исходников

```bash
cd /workspace/jackspawner
./gradlew clean build
# JAR будет в build/libs/JackSpawner-1.0.0.jar
```
