# EnderChestEC

**Расширяемый эндер-сундук для Paper 1.21.8**

Плагин добавляет возможность расширять эндер-сундук игрока за алмазы, увеличивая количество доступных слотов с базовых 27 до максимальных 81.

## 📋 Возможности

- **Команда `/ec`** - открывает виртуальный расширяемый эндер-сундук
- **Базовый размер** - 27 слотов (как обычный эндер-сундук)
- **Расширение** - +1 слот за 5 алмазов
- **Максимум** - 54 слота (6 строк по 9 слотов)
- **Персональное хранение** - у каждого игрока свой виртуальный сундук
- **Сохранение данных** - все предметы и прогресс сохраняются между сессиями

## 🚀 Установка

### Требования
- **Сервер**: Paper 1.21.8 или новее
- **Java**: 17 или новее

### Пошаговая установка

1. **Скачайте jar файл** из [Releases](../../releases)

2. **Поместите файл** `EnderChestEC-1.0.0.jar` в папку `plugins/` вашего сервера

3. **Перезапустите сервер** или выполните команду `/reload`

4. **Проверьте установку**:
   ```
   /plugins
   ```
   В списке должен появиться `EnderChestEC`

## ⚙️ Конфигурация

После первого запуска в папке `plugins/EnderChestEC/` создадутся файлы:

### config.yml
```yaml
# Базовое количество слотов
default-slots: 27
# Максимум слотов  
max-slots: 54
# Стоимость в алмазах за 1 слот
cost-per-slot: 5

# Сообщения
messages:
  not-enough-diamonds: "§cНедостаточно алмазов для разблокировки ячейки!"
  slot-unlocked: "§aЯчейка разблокирована! Потрачено: %cost% алмазов"
  max-slots-reached: "§eВы достигли максимального количества слотов!"
  locked-slot-name: "§c§lЗаблокировано"
  locked-slot-lore: 
    - "§7Нажмите, чтобы разблокировать"
    - "§7Стоимость: §e%cost% алмазов"
```

### player-settings.yml
Автоматически управляемый файл с данными игроков:
```yaml
players:
  "uuid-игрока":
    slots: 35
    inventory:
      0: {материал и данные предмета}
      1: {материал и данные предмета}
      # ... остальные слоты
```

## 🎮 Использование

### Команды

| Команда | Описание | Права |
|---------|----------|-------|
| `/ec` | Открыть расширяемый эндер-сундук | `enderchestec.use` |
| `/ec help` | Показать справку | `enderchestec.use` |
| `/ec info` | Информация о ваших слотах | `enderchestec.use` |
| `/ec reload` | Перезагрузить плагин | `enderchestec.admin` |

### Права доступа

| Право | По умолчанию | Описание |
|-------|--------------|----------|
| `enderchestec.use` | `true` | Использование команды /ec |
| `enderchestec.admin` | `op` | Админские функции |

### Как расширить сундук

1. Выполните команду `/ec`
2. В открывшемся сундуке найдите слоты с барьерами (🚫)
3. Кликните по заблокированному слоту
4. Если у вас есть 5+ алмазов, слот разблокируется
5. Алмазы автоматически спишутся из инвентаря

## 🐛 Устранение неполадок

### Плагин не загружается
- Проверьте версию Paper (нужна 1.21.8+)
- Проверьте версию Java (нужна 17+)
- Посмотрите логи сервера на ошибки

### Не сохраняются предметы
- Убедитесь, что у сервера есть права записи в папку `plugins/EnderChestEC/`
- Проверьте, не заполнен ли диск

### Команды не работают
- Проверьте права доступа: `/lp user <игрок> permission set enderchestec.use true`
- Убедитесь, что плагин активен: `/plugins`
