## JackSpawner (Paper 1.21.8)

Требования: Java 17+

Установка:
- Скачайте jar из релиза и положите его в `plugins/`
- Запустите сервер Paper 1.21.8

Команда:
- `/givejack` — выдать «Кирку Джека» (perm: `jackspawner.give`)

Описание логики:
- **Кирка Джека** — одноразовая золотая кирка со свечением. Она может ломать только блоки `Spawner`.
- При попытке ломать любой другой блок — ломание отменяется, кирка не тратится.
- Если ломаешь спавнер разрешённого моба, блок удаляется без обычного дропа. С шансом 20% дропается предмет `Spawner` с названием на русском и внутренним тегом с типом моба.
- Поддерживаемые типы: ZOMBIE, SKELETON, SPIDER, CAVE_SPIDER, WITCH, GUARDIAN, BLAZE, SILVERFISH.
- Названия предметов: «Спавнер зомби», «Спавнер скелетов», «Спавнер пауков», «Спавнер пещерных пауков», «Спавнер ведьм», «Спавнер стражей», «Спавнер ифритов», «Спавнер чешуйниц».
- Кирка всегда одноразовая: после попытки слома спавнера исчезает.
- Установка нашего кастомного предмета `Spawner` сразу создаёт активный спавнер с сохранённым типом моба.

Технические детали:
- API: `io.papermc.paper:paper-api:1.21-R0.1-SNAPSHOT`
- Java 17, Gradle (Kotlin DSL)
- Метаданные хранятся в `PersistentDataContainer`:
  - `jackspawner:is_jack_pickaxe` (Boolean) — отметка для «Кирки Джека»
  - `jackspawner:mob` (String) — тип моба у предмета-спавнера
- Без NMS, используется только Bukkit/Paper API
- Шанс дропа: `ThreadLocalRandom.current().nextDouble() < 0.2`
