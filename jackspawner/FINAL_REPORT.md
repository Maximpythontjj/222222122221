## 🎮 Функционал плагина

### Кирка Джека
- Золотая кирка с визуальным свечением (enchantment glint)
- Одноразовая (исчезает после использования)
- Ломает только спавнеры, остальные блоки защищены
- Лор с описанием функционала

### Поддерживаемые мобы (8 типов)
1. Зомби → "Спавнер зомби"
2. Скелет → "Спавнер скелетов"
3. Паук → "Спавнер пауков"
4. Пещерный паук → "Спавнер пещерных пауков"
5. Ведьма → "Спавнер ведьм"
6. Страж → "Спавнер стражей"
7. Ифрит → "Спавнер ифритов"
8. Чешуйница → "Спавнер чешуйниц"

### Логика работы
- 20% шанс дропа кастомного спавнера при ломании
- ThreadLocalRandom для генерации шанса
- PersistentDataContainer для хранения метаданных
- Автоматическая активация установленных спавнеров
