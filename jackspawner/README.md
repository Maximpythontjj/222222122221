# JackSpawner Plugin

Плагин "Кирка Джека" для Minecraft Paper 1.21.8 - добавляет одноразовую кирку, которая может ломать спавнеры определённых мобов с шансом 20% дропать кастомный спавнер.

## Версия

- **Paper**: 1.21.8
- **Java**: 21+
- **Версия плагина**: 1.0.0

## Требования

- Java 21 или выше
- Minecraft Paper Server 1.21.8

## Установка

1. Скачайте файл `jackspawner-1.0.0.jar` из релизов
2. Поместите jar файл в папку `plugins/` вашего Paper сервера  
3. Запустите или перезапустите сервер
4. Плагин готов к использованию!

## Команды

- `/givejack` - Выдать себе Кирку Джека (требует права `jackspawner.give`)

## Права доступа

- `jackspawner.give` - Позволяет получить Кирку Джека (по умолчанию только для операторов)

## Описание функционала

### Кирка Джека
- **Материал**: Золотая кирка с визуальным свечением
- **Прочность**: 1 использование (одноразовая)
- **Ограничения**: Может ломать только спавнеры, остальные блоки защищены

### Поддерживаемые мобы для спавнеров

Кирка может ломать спавнеры следующих мобов с шансом дропа:

- Зомби (ZOMBIE) → "Спавнер зомби"
- Скелет (SKELETON) → "Спавнер скелетов"  
- Паук (SPIDER) → "Спавнер пауков"
- Пещерный паук (CAVE_SPIDER) → "Спавнер пещерных пауков"
- Ведьма (WITCH) → "Спавнер ведьм"
- Страж (GUARDIAN) → "Спавнер стражей"
- Ифрит (BLAZE) → "Спавнер ифритов"
- Чешуйница (SILVERFISH) → "Спавнер чешуйниц"

### Логика работы

1. **Ломание спавнеров**: Киркой можно ломать только блоки спавнеров
2. **Шанс дропа**: 20% шанс получить кастомный спавнер при ломании
3. **Кастомные спавнеры**: Выпавшие спавнеры имеют русские названия и при установке автоматически активируются нужным мобом
4. **Одноразовость**: После любой попытки сломать спавнер кирка исчезает

## Технические особенности

- Использует PersistentDataContainer для хранения метаданных
- Совместим только с Paper API (не Bukkit/Spigot)
- Не использует NMS, только официальное API
- ThreadLocalRandom для генерации шанса дропа

## Сборка из исходников

```bash
git clone <repository-url>
cd jackspawner
./gradlew build
```

Собранный jar файл будет находиться в `build/libs/jackspawner-1.0.0.jar`
