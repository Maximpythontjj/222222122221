# JackSpawner Plugin v1.1.1

Minecraft плагин для Paper/Spigot серверов, добавляющий специальные "Кирки Джека" для добычи спавнеров.

## Описание

JackSpawner добавляет уникальную механику добычи спавнеров с помощью специальных кирок. Теперь доступны две версии кирок с разными шансами выпадения спавнеров и магазин для их покупки!

## 🆕 Новое в версии 1.1.1

- **Алмазная Кирка Джека** - версия с 50% шансом выпадения спавнера
- **Магазин кирок** - команда `/jackshop` для покупки кирок за алмазы
- **Улучшенная команда** `/givejack` с поддержкой параметров

## Возможности

### Типы кирок:
- **Золотая Кирка Джека** - 20% шанс выпадения спавнера
- **Алмазная Кирка Джека** - 50% шанс выпадения спавнера

### Особенности:
- **Одноразовое использование** - кирка исчезает после использования
- **Магазин кирок** - покупка за алмазы через GUI интерфейс
- **Поддержка различных мобов**: зомби, скелеты, пауки, пещерные пауки, ведьмы, стражи, ифриты, чешуйницы
- **Сохранение типа моба** при установке спавнера обратно

## Требования

- Paper или Spigot сервер версии 1.21+
- Java 21

## Установка

1. Скачайте последнюю версию плагина `JackSpawner-1.1.1.jar`
2. Поместите файл в папку `plugins` вашего сервера
3. Перезапустите сервер

## Команды

### `/givejack [тип]`
- **Описание**: Выдаёт Кирку Джека игроку
- **Разрешение**: `jackspawner.give` (по умолчанию: op)
- **Использование**: 
  - `/givejack` - выдает золотую кирку (20%)
  - `/givejack gold` или `/givejack 20` - золотая кирка (20%)
  - `/givejack diamond` или `/givejack 50` - алмазная кирка (50%)

### `/jackshop`
- **Описание**: Открывает магазин Кирок Джека
- **Разрешение**: `jackspawner.shop` (по умолчанию: все игроки)
- **Использование**: `/jackshop`
- **Цены**:
  - Золотая Кирка Джека (20%) - 15 алмазов
  - Алмазная Кирка Джека (50%) - 45 алмазов

## Разрешения

- `jackspawner.give` - Разрешение на получение Кирки Джека командой (по умолчанию: op)
- `jackspawner.shop` - Разрешение на использование магазина (по умолчанию: true)

## Как использовать

### Получение кирки:
1. **Через команду** (для администраторов): `/givejack [gold/diamond]`
2. **Через магазин** (для игроков): `/jackshop` и выберите нужную кирку

### Использование кирки:
1. Возьмите Кирку Джека в руку
2. Найдите спавнер моба
3. Разрушьте спавнер Киркой Джека
4. С определенным шансом (20% или 50%) спавнер выпадет как предмет
5. Подберите спавнер и установите его в нужном месте
6. Спавнер сохранит тип моба

## Особенности

- Кирка Джека работает **только** со спавнерами
- После использования кирка исчезает
- Выпавший спавнер сохраняет тип моба
- При установке спавнера тип моба восстанавливается
- В магазине автоматически проверяется наличие алмазов

