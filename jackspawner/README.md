# JackSpawner

Плагин для Minecraft Paper 1.21.8, добавляющий специальную кирку для добычи спавнеров.

## Требования

- **Версия сервера**: Paper 1.21.8
- **Java**: 17+

## Установка

1. Скачайте файл `JackSpawner-1.0.0.jar` из раздела Releases
2. Поместите JAR файл в папку `plugins/` вашего сервера
3. Запустите или перезапустите сервер

## Использование

### Команды

- `/givejack` - выдаёт Кирку Джека (требует права `jackspawner.give`)

### Права

- `jackspawner.give` - разрешение на использование команды `/givejack` (по умолчанию: op)

## Описание функционала

### Кирка Джека

- Зачарованная золотая кирка с визуальным свечением
- Одноразовое использование (исчезает после попытки сломать спавнер)
- Может ломать только спавнеры определённых мобов
- При ломании спавнера с шансом 20% выпадает сам спавнер с привязкой к мобу

### Поддерживаемые мобы

- Зомби (ZOMBIE)
- Скелет (SKELETON)
- Паук (SPIDER)
- Пещерный паук (CAVE_SPIDER)
- Ведьма (WITCH)
- Страж (GUARDIAN)
- Ифрит (BLAZE)
- Чешуйница (SILVERFISH)

### Механика работы

1. **Получение кирки**: используйте команду `/givejack`
2. **Ломание спавнера**: 
   - Кирка работает только со спавнерами
   - Попытка сломать другие блоки будет отменена
   - При ломании спавнера поддерживаемого типа с шансом 20% выпадет спавнер-предмет
3. **Установка спавнера**: полученный спавнер-предмет при установке сразу активируется нужным типом моба

## Сборка из исходников

```bash
./gradlew build
```

Скомпилированный JAR файл будет находиться в `build/libs/JackSpawner-1.0.0.jar`
