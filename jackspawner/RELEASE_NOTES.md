# JackSpawner v1.1.1

## ✨ Новое

- Добавлена Алмазная «Кирка Джека» с шансом 50%
- Добавлен магазин `/jackshop` (интерфейс в виде сундука):
  - Кирка Джека 20% — цена 15 алмазов
  - Кирка Джека 50% — цена 45 алмазов

## 🔧 Изменения

- Логика выпадения спавнера теперь читает шанс из предмета (PDC)
- Золотая кирка по умолчанию имеет шанс 20%

## 📦 Установка

1. Скачайте `JackSpawner-1.1.1.jar`
2. Поместите в папку `plugins/` вашего сервера
3. Перезапустите сервер

## 🎮 Использование

- `/givejack` — получить базовую (20%) Кирку Джека (требует `jackspawner.give`)
- `/jackshop` — открыть магазин кирок

## 🧩 Требования

- Paper 1.21.x
- Java 21+
