# JackSpawner v1.0.0

## 🎉 Первый релиз

Плагин JackSpawner для Minecraft Paper 1.21.8

### ✨ Функции

- **Кирка Джека** - специальная одноразовая золотая кирка с зачарованием
- **Добыча спавнеров** - 20% шанс получить спавнер при разрушении
- **Поддерживаемые мобы**:
  - Зомби
  - Скелет
  - Паук
  - Пещерный паук
  - Ведьма
  - Страж
  - Ифрит
  - Чешуйница

### 📦 Установка

1. Скачайте `JackSpawner-1.0.0.jar`
2. Поместите в папку `plugins/` вашего сервера
3. Перезапустите сервер

### 🎮 Использование

- `/givejack` - получить Кирку Джека (требует права `jackspawner.give`)

### 🔧 Требования

- Paper 1.21.8
- Java 21+
