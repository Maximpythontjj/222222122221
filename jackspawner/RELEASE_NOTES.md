# JackSpawner v1.1.1

## 🚀 Крупное обновление - Алмазная кирка и магазин!

### ✨ Новые функции

- **Алмазная Кирка Джека** - новая кирка с 50% шансом добычи спавнера
- **Магазин Кирки Джека** - новая команда `/jackshop` для покупки кирок за алмазы
- **Улучшенный GUI** - красивый интерфейс магазина с подсветкой предметов

### 🛠️ Изменения

- **Золотая Кирка Джека**: остался прежний 20% шанс, цена 15 алмазов
- **Алмазная Кирка Джека**: новая кирка с 50% шансом, цена 45 алмазов
- **Новая команда**: `/jackshop` - открывает магазин кирок
- **Улучшенная система**: поддержка нескольких типов кирок

### 📦 Установка

1. Скачайте `JackSpawner-1.1.1.jar`
2. Поместите в папку `plugins/` вашего сервера
3. Перезапустите сервер

### 🎮 Команды

- `/givejack` - получить золотую Кирку Джека (требует права `jackspawner.give`)
- `/jackshop` - открыть магазин Кирки Джека

### 🔧 Требования

- Paper 1.21.8
- Java 21+

# JackSpawner v1.0.0

## 🎉 Первый релиз

Плагин JackSpawner для Minecraft Paper 1.21.8

### ✨ Функции

- **Кирка Джека** - специальная одноразовая золотая кирка с зачарованием
- **Добыча спавнеров** - 20% шанс получить спавнер при разрушении
- **Поддерживаемые мобы**:
  - Зомби
  - Скелет
  - Паук
  - Пещерный паук
  - Ведьма
  - Страж
  - Ифрит
  - Чешуйница

### 📦 Установка

1. Скачайте `JackSpawner-1.0.0.jar`
2. Поместите в папку `plugins/` вашего сервера
3. Перезапустите сервер

### 🎮 Использование

- `/givejack` - получить Кирку Джека (требует права `jackspawner.give`)

### 🔧 Требования

- Paper 1.21.8
- Java 21+

### 📝 Изменения

- Первая версия плагина
- Полная реализация функционала согласно ТЗ
- Русская локализация названий спавнеров
