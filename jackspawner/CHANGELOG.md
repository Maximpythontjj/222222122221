# Changelog

## Version 1.1.1 (2025-09-16)

### Новые функции:
- **Алмазная Кирка Джека** - новая версия кирки с 50% шансом выпадения спавнера
  - Создана из алмазной кирки с зачарованной текстурой
  - Шанс выпадения увеличен до 50%
  - Сохраняет все функции золотой версии

- **Магазин Кирок Джека** - новая команда `/jackshop`
  - Открывает GUI-меню магазина
  - Золотая Кирка Джека (20% шанс) - цена 15 алмазов
  - Алмазная Кирка Джека (50% шанс) - цена 45 алмазов
  - Автоматическая проверка наличия алмазов у игрока
  - Удобный интерфейс покупки

### Улучшения:
- Обновлена команда `/givejack` - теперь поддерживает параметры:
  - `/givejack gold` или `/givejack 20` - выдает золотую версию (20%)
  - `/givejack diamond` или `/givejack 50` - выдает алмазную версию (50%)
  - Без параметров выдает золотую версию по умолчанию

- Улучшена система определения шанса выпадения - теперь шанс хранится в метаданных кирки

### Разрешения:
- `jackspawner.give` - разрешение на использование команды `/givejack` (по умолчанию: op)
- `jackspawner.shop` - разрешение на использование магазина `/jackshop` (по умолчанию: true)

## Version 1.0.0 (Initial Release)
- Золотая Кирка Джека с 20% шансом выпадения спавнера
- Команда `/givejack` для получения кирки
- Поддержка спавнеров: зомби, скелетов, пауков, пещерных пауков, ведьм, стражей, ифритов, чешуйниц
- Одноразовое использование кирки
